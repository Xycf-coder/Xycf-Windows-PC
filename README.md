## 中文版
# Windows 电脑工具箱
一个用 C++ 编写的 Windows 小工具合集，适用于 Dev-C++ 编译运行。
## 功能列表
- 磁盘扫描（快速检查磁盘状态）
- 强制重启
- 强制关机
- 控制台调色（更改命令行颜色）
- 查看本机信息（系统版本、CPU、内存等）
## 如何使用
1. 用 Dev-C++ 打开 `.cpp` 文件。
2. 按 F11 编译并运行。
3. 根据菜单提示选择对应功能。
## 运行环境
- Windows 7 / 8 / 10 / 11
- Dev-C++ 5.11 或更高版本

# Windows PC Toolbox
A C++ utility collection for Windows, compiled with Dev-C++.
## Features
- Disk scanner (quick disk status check)
- Force restart
- Force shutdown
- Console color changer
- System information viewer (OS version, CPU, RAM, etc.)
## How to Use
1. Open the `.cpp` file with Dev-C++.
2. Press F11 to compile and run.
3. Follow the menu to select the desired function.

- Windows 7 / 8 / 10 / 11
